### How do I know which action to use?
Often multiple actions will be relevant, choose whatever works best for you! If you suspect it isn't correct, try to use it in a sentence. This isn't a rule, but can be a useful tool.
Some common examples of this is:
- I want to *sneak* past the guard
- I want to *force* the door open
- I want to *perform* a song
- I want to *search* for clues
